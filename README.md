Расширение конфигурации [1С:Конвертация данных 3.1](https://its.1c.ru/db/metod8dev#content:5846:hdoc), адаптирующее типовой механизм обмена под работу с произвольным XDTO в [1С: Адаптер Kafka](https://github.com/ShadobaAI/kafka-adapter).

## Назначение

При использовании обработчика типа **«КД 3.1»** адаптер передаёт данные через модуль обмена, сгенерированный в 1С:Конвертации данных. Стандартный механизм КД 3.1 ориентирован на формат **EnterpriseData**. Данное расширение позволяет использовать вместо него **произвольный XDTO-пакет** — то есть контракт данных, специфичный для вашей интеграции.
